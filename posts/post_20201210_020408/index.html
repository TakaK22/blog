Visual Studio CodeでPlantUmlのプレビューができなくなった

December 10, 2020

久しぶりにPlantoUMLを触って見ようと思ってサンプルコードを実行してみたらプレビューが出来なくなっていた

version

  • macOS: Big Sur 11.0.1
  • Visual Studo Code:1.51.1
  • PlantUML: 2.14.0

PlantUMLプレビューを行うと以下のエラーが出た

javaが見つかりません!
javaをインストール済みの場合は、PATH環境変数にjava binへのパスを追加してください。
または"plantuml.java "でjava実行可能パスを設定してください。

vscode の設定を開きplantuml.javaで検索すると未設定だった

ターミナルからJavaコマンドが使えるか確認してみる

% java --version

OpenJDK 64-Bit Server VM (build 15.0.1+9, mixed mode, sharing)

==> Caveats
For the system Java wrappers to find this JDK, symlink it with
  sudo ln -sfn /usr/local/opt/openjdk/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk.jdk

openjdk is keg-only, which means it was not symlinked into /usr/local,
because it shadows the macOS `java` wrapper.

If you need to have openjdk first in your PATH run:
  echo 'export PATH="/usr/local/opt/openjdk/bin:$PATH"' >> ~/.zshrc

  export CPPFLAGS="-I/usr/local/opt/openjdk/include"
